VMware中为Fedora 23安装VMware Tools

  1. 添加163镜像源。

    # cd /etc/yum.repos.d
# wget http://mirrors.163.com/.help/fedora-163.repo
# wget http://mirrors.163.com/.help/fedora-updates-163.repo

  2. 配置dnf自动选择最快的镜像源

    # vim /etc/dnf.conf
#在文件末尾添加一行
fastestmirror=true
#wq保存退出

  3. 生成缓存

    # dnf clean all
# dnf makecache

  4. 检查更新 && 升级更新

    # dnf check-update
# dnf upgrade

  5. 安装编译环境

    # dnf install gcc glibc-headers kernel-devel kernel-headers make perl

  6. 安装Open-vm-tools

    # dnf install open-vm-tools

  7. 挂载镜像

    # mkdir /mnt/cdrom
# mount /dev/cdrom /mnt/cdrom

  8. 解压 && 安装

    # tar -C ~/ -zxvf /mnt/cdrom/VMwareTools-10.0.1-3160059.tar.gz
# ~/vmware-tools-distrib/vmware-install.pl

    一路回车安装完毕。
